# 🚀 SyllogosHub - Master Task List (V2.0)

## 1. Ασφάλεια & Πρόσβαση (Sensitive Data First) 🔒
- [ ] **Supabase Auth:** Υλοποίηση Login (Email/Password).
- [ ] **Role-Based Access (RBAC):** 
    - **Admin (Πρόεδρος/Ταμίας):** Πλήρης πρόσβαση (Οικονομικά, Settings).
    - **Editor (Δ.Σ./Δάσκαλοι):** Μητρώο, Seating, Παρουσίες.
    - **Public (Επισκέπτης):** Μόνο Ημερολόγιο & Ανακοινώσεις.
- [ ] **Sensitive Logic:** Κλείδωμα σελίδας "Οικονομικά" μόνο για Admin.

## 2. Αναβάθμιση Μητρώου Μελών 👥
- [ ] **Multi-Department:** Ένα μέλος να ανήκει σε πολλά τμήματα (π.χ. Χορευτικό + Λύρα).
- [ ] **Board Members:** Πεδίο "Θέση στο Δ.Σ." και Checkbox "Μέλος Συμβουλίου".
- [ ] **Advanced Filters:** Φιλτράρισμα ανά στήλη, τμήμα, ρόλο και οικονομική κατάσταση.
- [ ] **Import/Export:** Μαζική εισαγωγή από Excel και εξαγωγή λίστας μελών.

## 3. Έξυπνο Seating Plan (Event Engine) 🪑
- [ ] **Dynamic Capacity:** Κουμπιά `+/-` πάνω στο τραπέζι για αλλαγή θέσεων (π.χ. από 8 σε 10).
- [ ] **Table Chaining:** Αυτόματη διαχείριση μεγάλης παρέας (π.χ. παρέα 15 ατόμων -> κατάληψη τραπεζιών 14 & 15).
- [ ] **Guest List Management:** 
    - Σύνδεση μελών από το Μητρώο σε συγκεκριμένη θέση.
    - Χειροκίνητη προσθήκη ονομάτων για μη-μέλη (επισκέπτες).
- [ ] **Batch Create:** Μαζική δημιουργία τραπεζιών (π.χ. 1-50).

## 4. Κεντρικό Ημερολόγιο (Calendar) 📅
- [ ] **Interactive Calendar:** Προβολή όλων των δραστηριοτήτων (Μαθήματα, Γ.Σ., Εκδηλώσεις).
- [ ] **Categories:** Διαφορετικά χρώματα ανά τύπο (π.χ. Μπλε=Μαθήματα, Κόκκινο=Εκδηλώσεις).
- [ ] **Public View:** Σελίδα ημερολογίου προσβάσιμη από όλους χωρίς Login.

## 5. Οικονομικά & Εκτυπώσεις 💶
- [ ] **Financial Templates:** Προκαθορισμένα ποσά (π.χ. Συνδρομή 10€, Μάθημα 15€).
- [ ] **PDF Engine:** Αυτόματη έκδοση απόδειξης πληρωμής και Λίστας Εισόδου (Alphabetical).
- [ ] **Event Info:** Πεδία για Χορηγούς, Κόστος Πρόσκλησης και συνολικό ταμείο εκδήλωσης.

## 6. Branding & Επικοινωνία (White Label) 🎨
- [ ] **Theme Settings:** Σελίδα για Logo και Χρώματα (Μπορντό/Μαύρο στυλ Ένωσης Κρητών).
- [ ] **Messaging:** Σύνδεση με Viber/WhatsApp για μαζικές ενημερώσεις και επιβεβαιώσεις τραπεζιών.
